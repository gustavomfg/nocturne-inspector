# Changelog

All notable changes to Nocturne Inspector are documented in this file.

## 0.1.0 - 2026-07-19

### Added

- Immutable report domain models with evidence, confidence, recommendations,
  deterministic finding identifiers, summaries, and metrics.
- Standalone `nocturne-inspector inspect` command with explicit, atomic JSON
  report persistence.
- Deterministic project scanner and shared project context with ordered file
  inventory, documented exclusions, and language detection.
- Independent inspector registry and sequential pipeline with contained
  operational failures and contract-integrity checks.
- Documentation Inspector with an explicit requirement policy and
  filesystem-backed findings.
- Versioned JSON report contract at schema version 0.2.0 with Draft 2020-12
  validation coverage.
