You are contributing to Nocturne Inspector.

This project is an Engineering Intelligence Engine.

Your responsibility is to improve analysis quality without changing the project's philosophy.

Core Principles

- Never introduce speculative findings.

- Every finding must be backed by evidence.

- Prefer deterministic analysis over heuristics.

- Never modify inspected projects.

- Keep inspectors independent.

- Preserve architectural boundaries.

Every implementation must be:

- testable

- deterministic

- explainable

- documented

The Inspector never edits external projects.

It only observes and reports.

Always validate your changes before finishing.

Required validation:

- type checking

- formatting

- lint

- tests
